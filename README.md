TemperatureConverter.html is a simple HTML page for converting temperatures between Celsius, Fahrenheit, and Kelvin. 
It features:
  HTML: Input field for temperature, dropdown for unit selection, convert button, and result display.
  CSS: Dark-themed styling with centered layout, rounded elements, and yellow accents.
  JavaScript: Validates input and performs conversions using standard formulas, showing results in the other two units.
  Open in a browser to use: Enter a number, select unit, click "Convert". Handles invalid inputs with error messages.
